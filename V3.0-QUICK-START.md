## ğŸ”§ TROUBLESHOOTING {#troubleshooting}

### âŒ Nothing loads or looks broken
**Solution**:
1. Check browser console (F12 â†’ Console tab)
2. Look for red error messages
3. Clear browser cache (Ctrl+Shift+Delete)
4. Refresh page (Ctrl+F5)

### âŒ Animations don't work
**Solution**:
1. Check if `prefers-reduced-motion` is enabled
2. Verify CSS file loaded (DevTools â†’ Network tab)
3. Check browser compatibility
4. Try different browser

### âŒ Dark mode doesn't save
**Solution**:
1. Check if localStorage is enabled
2. Verify browser privacy settings
3. Try incognito/private mode
4. Clear browser data

### âŒ Chatbot doesn't respond
**Solution**:
1. Try keywords: "admission", "academic", "fee", "contact"
2. Type "help" for default response
3. Check JavaScript console for errors
4. Verify script-v3.js is loaded

### âŒ Tilt effect doesn't work on mobile
**Solution** (intentional):
- Tilt is disabled on touch devices for performance
- Can be re-enabled in code if needed
- Works on desktop/laptop with mouse

### âŒ Mobile menu stuck open
**Solution**:
1. Click hamburger button again
2. Click any menu link to close
3. Press ESC key
4. Refresh page

### âŒ Form won't submit
**Solution**:
1. Check all required fields are filled (red border)
2. Verify email format is correct
3. Check browser console for JS errors
4. Ensure phone number (if required) is valid
