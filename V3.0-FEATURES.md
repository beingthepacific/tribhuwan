Always use variables instead of hardcoded values:


### Common Issues & Solutions

**Animations Not Working**
- Check if `prefers-reduced-motion` is enabled
- Verify CSS file is linked correctly
- Check browser DevTools for CSS errors

**Chatbot Not Responding**
- Verify JS file is loaded (check console)
- Check that input text matches keywords
- Ensure browser console shows no errors

**Dark Mode Not Persisting**
- Check if localStorage is enabled
- Verify theme is being saved to localStorage
- Clear browser cache and try again

**Tilt Effects Not Working on Mobile**
- This is intentional (touch doesn't support hover)
- Can be enabled by removing mobile check if needed
